# 升學指南針 Admission Compass V2

這是 GitHub + Vercel 正式版骨架。

## 上傳 GitHub 時請注意

解壓縮後，請進入 `admission_compass_v2_github` 資料夾。

GitHub 根目錄必須直接看到：

- app
- components
- data
- public
- package.json
- tsconfig.json
- next.config.mjs
- next-env.d.ts
- README.md

不要把 app 裡面的 page.tsx、globals.css 拉出來單獨上傳。

## Vercel

從 Vercel 匯入 GitHub repository。
Framework Preset 選 Next.js。
直接 Deploy。
